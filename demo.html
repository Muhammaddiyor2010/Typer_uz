How to Test the Multiplayer Features

1

Open Multiple Browser Windows

Open the main app in 2-3 different browser windows or tabs to simulate multiple players.

2

Create a Room

In the first window, click "Create Room" to become the room creator. Copy the room code.

3

Join the Room

In other windows, click "Join Room" and enter the room code to join as regular players.

4

Test Settings

Try changing settings in the room creator window - notice how joined players can't change them.

5

Start Typing

Start the typing test and watch the live progress indicators and cursors in real-time!
